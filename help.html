Argumentation Tool Help

Overview

This tool enables you to build and manipulate a graphical structure consisting of text-containing nodes and arrows that represents a logical argument. Specifically, it uses the syntax of the Goal Structuring Notation, which provides the following types of node: Goal (rectangular node), Strategy (parallelogram), Solution (circle), Context (rounded rectangle), Assumption (ellipse) and Justification (ellipse). Structures can be saved to file and exported as PNG images.

Building an Argument Structure

Argument structures are built by creating, editing and deleting nodes, and creating and deleting connections between nodes. These functionalities are accessed either from the Build menu or by right clicking an existing node with the mouse. Selecting one of these functionalities displays a popup, described below.

Create Node

Here you can select the Type of the node: Goal (rectangular node), Strategy (parallelogram), Solution (circle), Context (rounded rectangle), Assumption (ellipse) and Justification (ellipse). The text to be displayed in the node is provided by the Title and the Content. The Content allows you to add simple formatting: bold, italic, and text colour, and to embed a link via a standard web URL.

Edit Node

If accessing from the Build menu, you can select the node you wish to edit from a drop-down list of node titles. The edit dialog allows you to edit the Title and Content of the node.

Delete Node

If accessing from the Build menu, you can select the node you wish to delete from a drop-down list of node titles. The node will be removed from the argument structure along with any connections to or from the deleted node.

Create Connection

Creates an arrow from a parent node to a child node. The type of arrowhead (filled or unfilled) is chosen automatically. If accessing from the Build menu, you can select the parent and child ends of the arrow from drop-down boxes. If accessing from right clicking the node, you can choose whether the node is the parent or child end of the connection and select the other from a drop-down box.

Delete Connection

Removes from the structure the connection chosen from a drop-down box.

Manipulating an Argument Structure

The View menu provides the ability to zoom in and out, and to centre the structure in the window.

Nodes in the argument structure can be individually moved using the computer mouse by left clicking and dragging the node. Left mouse clicking and dragging on areas outside of the nodes will drag the entire structure.

Additionally the following functionality can be accessed by right-clicking a node with the mouse:

  • Collapse: any node which is a parent node (there is a connection from that node to another) can be collapsed to hide all the nodes below it in the structure. If collapsed, a node will display a filled diamond in place of the collapsed structure.
  • Expand: any node which has been collapsed (displays a filled diamond) will display the Expand item instead of Collapse. Expanding the node will reveal the collapsed structure.
  • Mark Undeveloped: right-clicking on a Goal node which has no child connections allows access to this option. If marked undeveloped, a goal node will have an unfilled diamond placed where a child connection would be. This is part of the Goal Structuring Syntax.

Load, Save and Export

The File menu enables you to open and save a file that describes the argument structure, or to export the structure as a PNG image. The menu items under the File menu have the following functionality:

  • Open File: will open a previously saved structure from a location on your local machine.
  • Save File: will save your structure to a file on your local machine. The saved file is in the XML format.
  • Export as PNG: will save a PNG image representation of your structure to a file on your local machine.
